# QuickPerf 1.0.0

## QuickPerf

**QuickPerf is a testing library for Java to quickly evaluate and improve some performance-related properties.**

The release candidate distributions are used by several companies, among them _Renault Digital_ and _Fujitsu Portugal_.

QuickPerf also allowed [benching the Maven heap allocation](https://github.com/quick-perf/maven-test-bench).

### :point_right: [**Lean more about QuickPerf**](https://github.com/quick-perf/doc/wiki/QuickPerf)


## Project story
QuickPerf is an OSS project licensed under the [Apache License, Version 2.0](https://www.apache.org/licenses/LICENSE-2.0).

It was created by [Jean Bisutti](https://twitter.com/jean_bisutti) who presented preliminary versions to _open space conferences_ (also called _unconferences_):  [Socrates France](https://socrates-fr.github.io) 2017 & 2018, [JAlba](https://jalba.scot/) 2018 & 2019, and [JCrete](https://www.jcrete.org/) 2018. Several people gave positive feedback, suggestions, and encouragement. _Many thanks to all of them!_

Several contributors joined the project. _Many thanks to all of the QuickPerf contributors!_
